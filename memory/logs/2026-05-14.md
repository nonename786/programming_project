# 证明：积分 $\int_0^{+\infty} \frac{y\,dx}{(x^2 + y^2)^{3/2}}$ 在 $1 \leq y \leq 4$ 上一致收敛

---

## 方法一：Weierstrass M-判别法（优函数判别法）

**步骤 1：寻找优函数**

对于 $x \geq 0$，$1 \leq y \leq 4$，我们有：

$$\left| \frac{y}{(x^2 + y^2)^{3/2}} \right| = \frac{y}{(x^2 + y^2)^{3/2}}$$

由于 $y \leq 4$ 且 $y^2 \geq 1$，可得：

$$\frac{y}{(x^2 + y^2)^{3/2}} \leq \frac{4}{(x^2 + 1)^{3/2}} \triangleq M(x)$$

**步骤 2：验证优函数的积分收敛**

$$\int_0^{+\infty} M(x)\,dx = \int_0^{+\infty} \frac{4}{(x^2 + 1)^{3/2}}\,dx$$

令 $x = \tan\theta$，则 $dx = \sec^2\theta\,d\theta$，$(x^2+1)^{3/2} = \sec^3\theta$：

$$= 4\int_0^{\pi/2} \frac{\sec^2\theta}{\sec^3\theta}\,d\theta = 4\int_0^{\pi/2} \cos\theta\,d\theta = 4[\sin\theta]_0^{\pi/2} = 4 < +\infty$$

**步骤 3：应用 Weierstrass M-判别法**

由于存在与 $y$ 无关的函数 $M(x)$ 使得 $|f(x,y)| \leq M(x)$，且 $\int_0^{+\infty} M(x)\,dx$ 收敛，根据 **Weierstrass M-判别法**，原积分在 $1 \leq y \leq 4$ 上**一致收敛**。

---

## 方法二：直接计算法（验证）

令 $x = y\tan\theta$，则 $dx = y\sec^2\theta\,d\theta$：

$$\int_0^{+\infty} \frac{y\,dx}{(x^2 + y^2)^{3/2}} = \int_0^{\pi/2} \frac{y \cdot y\sec^2\theta}{(y^2\tan^2\theta + y^2)^{3/2}}\,d\theta$$

$$= \int_0^{\pi/2} \frac{y^2\sec^2\theta}{y^3\sec^3\theta}\,d\theta = \frac{1}{y}\int_0^{\pi/2} \cos\theta\,d\theta = \frac{1}{y}$$

积分结果为 $I(y) = \frac{1}{y}$，在 $[1,4]$ 上连续有界，这也说明积分一致收敛。

---

**结论：** 该含参变量反常积分在 $1 \leq y \leq 4$ 上一致收敛。
